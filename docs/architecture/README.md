## Tecnologías

| Capa | Tecnología | Propósito |
|------|------------|-----------|
| Frontend | Streamlit | Dashboard interactivo |
| API | FastAPI | REST API |
| Scheduler | APScheduler | Pipeline automático |
| Database | TimescaleDB | Almacenamiento temporal |
| Cache | Redis | Cache de features |
| Monitoring | Prometheus + Grafana | Observabilidad |
| ML | LightGBM + SHAP | Modelos predictivos |
| Container | Docker | Orquestación |
